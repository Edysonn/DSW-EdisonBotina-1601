Descubre los selectores CSS


¿Qué son?


Comprendamos un poco de fundamento. La CSS (coding style sheets)son hojas de estilo; archivos responsables de configurar la apariencia de la página web. Todas las paginas de un mismo sitio comparten la misma hoja de estilos. A esta hoja de estilos se la conoce como un archivo con extensión .css y estas definen una o más reglas, las cuales se aplicarán a los elementos escogidos en dicha regla. Estas reglas se componen de dos partes: Selector: evidencia a qué elementos se aplicará la regla. Se puede aplicar más separándolo con una coma (,).


Lista de declaraciones: son los estilos que se van a aplicar a los elementos que cumplen la regla. Su orden es esta: propiedad:valor, y se separa con un punto y coma. Los selectores css son parte de unas reglas, los cuales se indica a qué elementos se aplicará dicha orden, definiendo el estilo que se quiere dar a dichos elementos en CSS.

Selector universal



este selector se define como (*) y se aplica a todos los elementos de dicha página.



_____________________________

_____________________________

Selector de etiqueta HTML



se indica el estilo que se va a un elemento o etiqueta concreto (se quita el < y > de las etiquetas).



_____________________________

_____________________________

Selector de clase



A los elementos HTML se les puede añadir un atributo class el cual se indica las clases a la que pertenece el elemento. Entonces a esa clase se la permite tratar de una forma particular, así en CSS y también en JavaScript. En CSS se puede aplicar una regla a todos los elementos que sean de determinada clase.



_____________________________

_____________________________

Selector de identificador



Se puede aplicar el estilo único al único elemento de de una página, el cual tiene un determinado identificador (especificado con el atributo id). Se especifica con # más el nombre del identificador.
Para este caso se crea un relleno superior al elemento y un background-color al único párrafo que se identifica como dias.

Selectores descendents y adyacentes



Permiten seleccionar elementos contenidos dentro de otros de acuerdo a la estructura del árbol del DOM (Model de Objeto del Documento (Interfaz de programación para los documentos HTML y XML)). Cuando se separa dos selectores por un espacio en blanco, se aplica la regla a todos los descendientes del primer selector.


Este selector se aplica a todos los elementos p que son descendientes de un elemento section (pueden se descendientes de cualquier nivel).

Selectores de atributos


Se utilizan para seleccionar elementos en función de sus atributos. Hay cuatro tipos:


[atributo]: elementos que tienen definido el atributo llamado atributo, independiente de su valor.


[atributo=valor]: elementos que tienen un atributo llamado atributo con el valor especificado.


[atributo-=valor]: elementos que tienen un atributo llamado atributo y al menos uno de los valores del atributo es el valor especificado.


[atributo^=valor]: elementos que tienen un atributo llamado atributo y cuyo valor empieza por el valor especificado.


[atributo$=valor]: elementos que tienen un atributo llamado atributo y cuyo valor termina por el valor especificado.

Las pseudoclases


Hay selectores especiales donde se especifica elementos que tienen unas determinadas propiedades. Son como clases "virtuales", ya que no se especifican en el HTML, aunque representan las posiciones o estados en los que se puede encontrar un elemento mientras se visualiza en el navegador.


En el ejemplo de al lado, podemos ver una aplicación de un valor a un elemento que solo es el primer hijo de su padre, utilizando la pseudoclase first-child. Entonces se aplicará al primer elemenro li de una lista de viñetas ul, lo cual aparece en color verde.


Existen más pseudclases que ayudan a seleccionar a un hermano concreto de entre dos hijos de un elemento los cuales son:


:nth.child(n): representa el n-ésimo hermano. Permite incluir una forma más compleja, como :nth.child(even) o :nth.child(odd), que representa dos hijos que ocupan posiciones pares o impares (por ejemplo, hacer que las filas impares de una tabla tengan distinto color de fondo).


nth-last-child(n): lo mismo que el anterior, pero contando desde el final.


Estas anteriores se utilizan usualmente para crear tablas cebreadas, donde las filas pares e impares tienen apariencia diferente.


Existen también las pseudoclases dinámicas, donde indican el estado de los elementos cuando el usuario interactúa con ellos:


:hover: se utiliza cuando se quiere establecer el estilo cuando el usuario coloca el mouse sobre el elemento.


:active: se utiliza para establecer el estilo cuando el usuario pincha sobre el elemento o mantiene el botón del mouse pulsado sobre el.


:focus: se utiliza para establecer el estilo cuando el elemento tiene el foco.


Son muy usuales las pseudoclases relacionadas con los enlaces:


:link: establece el estilo del enlace cuando aún no se ha visitado o cuando el usuario aún no lo ha pulsado.


:hover: establece el estilo cuando el usuario coloca el mouse sobre el elemento.


:active: establece el estilo cuando el usuario pincha sobre el enlace.


a:visited: hace que el estilo de un enlace ya visitado por el usuario.


En el ejemplo se observará lo siguiente: los enlaces del documento aparecen sin subrayar y en color verde. Al situar el mouse sobre ellos, pasan a azul y, una vez visitados, se muestran en rojo.
