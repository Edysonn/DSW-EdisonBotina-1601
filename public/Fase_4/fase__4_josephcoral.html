Quiz CSS-Grid Joseph Coral



1. ¿Cuál de las siguientes propiedades se utiliza para definir el número de columnas en una cuadrícula CSS Grid?


a) grid-template-columns
b) grid-column
c) grid-auto-columns
d) grid-area



2. ¿Cuál es la función de la propiedad grid-template-areas en CSS Grid?


a) Define el tamaño de las áreas en la cuadrícula.
b) Establece el número de filas en la cuadrícula.
c) Define las áreas nombradas y su ubicación en la cuadrícula.
d) Determina el orden de los elementos en la cuadrícula.



3. ¿Cuál de las siguientes propiedades se utiliza para establecer el tamaño de una fila o columna en CSS Grid?


a) grid-gap
b) grid-template-rows
c) grid-row
d) grid-auto-rows



4. ¿Qué función se utiliza para crear un tamaño flexible en CSS Grid que se repite varias veces?


a) repeat()
b) calc()
c) minmax()
d) fit-content()



5. ¿Cuál de las siguientes opciones es una característica clave de CSS Grid?


a) Soporte solo para navegadores modernos.
b) Capacidad de crear diseños bidimensionales.
c) No es compatible con diseños responsivos.
d) Reemplazo de flexbox en todas las situaciones de diseño.
